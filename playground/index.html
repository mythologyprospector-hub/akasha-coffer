What this is

This is not a live banking app. It is a local Akasha treasury playground: a truthful place to record support, watch flow, and think clearly about what sustains the happy place.

CLI

Use the coffer CLI from the repo root:

python cli/coffer.py status
python cli/coffer.py add-income --source "buymeacoffee" --amount 5 --note "test"
python cli/coffer.py add-expense --category "api" --amount 20 --note "monthly cost"
python cli/coffer.py snapshot --label "after-support-wave"
